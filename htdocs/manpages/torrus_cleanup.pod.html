NAME

cleanup - Cleans up Torrus web session data.

SYNOPSIS

torrus cleanup

DESCRIPTION

This command cleans out the expired (older than 2 days) Torrus web interface session data.

FILES

/var/torrus/session_data/store
Torrus session data storage directory.

/var/torrus/session_data/lock
Torrus session data lock directory.

SEE ALSO

torrus(1)

NOTES

See more documentation at Torrus home page: http://torrus.org
