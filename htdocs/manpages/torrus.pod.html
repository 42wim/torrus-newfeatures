NAME

torrus - Commandline wrapper for Torrus utilities

SYNOPSIS

torrus command [options...]

DESCRIPTION

All Torrus utilities use this common command-line wrapper, which simply passes all arguments to the appropriate utility.

Accepted commands:

acl | acledit
cleanup
clc | clearcache
collector
compile | compilexml
ci | configinfo
configsnapshot
discover | devdiscover
genddx
genlist
install_plugin
monitor
si | schedulerinfo

This list may be extended by Torrus plugins.

OPTIONS

Refer to approproate manpages for options for each Torrus command.
